# Prohelp Release Notes: v0.1.0 (2026-05-27)

This is the initial public release of **Prohelp** (Progressive & Professional Help), a complete specification and reference library for implementing hierarchical, progressive command-line help pages.

---

## What is Prohelp?

Prohelp solves the terminal "wall of text" problem by dividing command-line documentation into a logical hierarchy of sections and options. Rather than dumping hundreds of flags at once, Prohelp organizes data conceptually and allows users to progressively traverse the hierarchy using raw command-line tokens or a rich full-screen browser.

---

## Detailed Features in this Release

### 1. The Core Dlang Interceptor Engine
* Bundles `prohelp.parser` to read SDLang (`help.sdl`) configurations, build abstract syntax trees, and enforce sliding-scale line budgets at parse time (Level 0 <= 20 lines; Level 1 <= 40 lines; Level 2+ <= 60 lines).
* Integrates `prohelp.intercept` which exposes a 1-line integration API for developers (`if (prohelp.intercept(args)) return;`).

### 2. Hybrid Option Suffixes & Interceptions
* Supports the `?:[locale][,mode]` hybrid console parameter (e.g. `?:de,i` or `?:text`). 
* Isolates help system parameters (like locales and output formats) from actual subcommand navigation tokens.
* Intercepts `help`, `?`, `--help`, `-h`, and `--?` triggers automatically.

### 3. Unicode Box-Drawing Visual Zoning (Text Mode)
* Defaults to a static, line-by-line printout framed with Unicode box-drawing glyphs (`┌`, `─`, `┐`, `│`, etc.).
* Creates structured visual "information cards" on the console.
* Fully compatible with standard UNIX pipes (`|`), redirections (`>`), and plain text outputs.

### 4. Interactive Viewport Browser (Interactive Mode)
* launches an opt-in full-screen TUI terminal viewport (`?:i` or `?:interactive`).
* Implements:
  * **Real-time Alphanumeric Filters**: Typing immediately narrows down visible subcategories.
  * **Shift-Toggled Cell Highlighting**: Pressing and releasing `Shift` switches to Highlight Mode, where the arrow keys select exact grids of text.
  * **Markdown Copying (`Ctrl+C`)**: Converts highlighted console characters to clean, structured Markdown and copies it directly to the system clipboards (supports native Win32 Unicode clipboard & POSIX pbcopy/xclip/xsel).
  * **Safe Exit (`Ctrl+Esc`)**: Restores original shell console flags and safely terminates.

### 5. Wildcard Subtree Globbing (`*`)
* Appending `*` or `all` in Text Mode dumps the whole section subtree recursively.
* Enables AI agents and scripts to retrieve highly targeted, compact, structured subsets of commands on demand without context window clutter.
